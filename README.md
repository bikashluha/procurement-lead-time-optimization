# Procurement Lead Time Optimization

## Problem
Cycle time from PR to PO was 5 days

## Objective
Reduce cycle time by identifying bottlenecks using SQL

## Approach
- Calculated cycle time
- Identified stage-wise delays
- Found bottlenecks
- Validated improvements

## Results
- Reduced cycle time from 5 → 3 days
- Improved efficiency by ~40%

- SQL
- Excel / Power BI
